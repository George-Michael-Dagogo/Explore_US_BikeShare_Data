In order to use the web app, you need to install streamlit:
. open your Anaconda power shell prompt and "pip install streamlit".
. after installation make sure the team_matplotlib.py file and csv files are in the same directory as stated in the Anaconda powershell prompt "C:\Users\HP>" or whichever directory is displayed in your cmd shell.
. to run the team_matplotlib.py file on a server simply type "streamlit run team_matplotlib.py" in your powershell prompt and it will automatically open your default browser and commence the web app.
